# Introduction to Git and GitHub

[![License](https://img.shields.io/badge/License-Apache%202.0-blue.svg)](LICENSE)

This repository contains sample code for an Introduction to Git and GitHub course. It incorporates basic calculators to demonstrate version control workflows.

## Features

This project includes two main scripts:

### 1. Simple Interest Calculator (`simple-interest.sh`)
A Bash script that calculates simple interest given the principal amount, annual rate of interest, and time period in years.

**Formula:**
`Simple Interest = (Principal * Time * Rate) / 100`

**How to run:**
```bash
bash simple-interest.sh
```

### 2. Compound Interest Calculator (`compound_interest.py`)
A Python script that calculates yearly compound interest.

**Formula:**
`Compound Interest = Principal * (1 + Rate/100)^Time`

**How to run:**
```bash
python compound_interest.py
```
